= LENSKIT-VERSION(1)
:doctype: manpage

== NAME

lenskit-version - print LensKit version info.

== SYNOPSIS

*lenskit* [_GLOBAL OPTIONS_] *version*

== DESCRIPTION

The *version* command prints the current version of LensKit and exits.

This subcommand takes no arguments.
